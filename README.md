# Pricing-UI-with-Bootstrap

A simple and responsive pricing section built using **Bootstrap 5**.  
This mini project demonstrates the use of Bootstrap grid system, cards, buttons, and utility classes to create a clean pricing UI.

# Project Overview

The project displays a **pricing page** with three different plans:
Free
Pro
Enterprise

Each plan is presented using Bootstrap cards and is fully responsive across different screen sizes.

This project is created as a **mini frontend project** to showcase Bootstrap styling and layout capabilities.

# Technologies Used

HTML5
Bootstrap 5 (via CDN)

# Features

Responsive grid layout
Bootstrap card components
Utility classes for spacing and alignment
Mobile-friendly design
Clean and minimal UI
